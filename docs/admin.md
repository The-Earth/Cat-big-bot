# Admin

admin 用户由机器人操作者直接授权，是可以管理投票的用户。

## 权限
* 可以使用 [trusted](trusted.md) 用户可使用的指令
* 可以使用 [voter](voter.md) 用户可使用的指令
* 于群组中可以使用：
    * `/set_voter`
    * `/list_voter`
    * `/unset_voter`
    * `/init_poll`
    * `/start_poll`
    * 点按关闭投票按钮
* 于私聊中可以使用：
    * `/set_voter`
    * `/unset_voter`
